# Ma'My House Services

Site vitrine Next.js pour **Ma'My House Services**.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Accueil |
| `/services` | Tous nos services |
| `/blog` | Articles et conseils |
| `/contact` | Formulaire et coordonnées |

## Démarrage

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000).
